## ✨ Architectural Features & Innovations

- **🌀 Kinetic 3D Physics Engine:** Interactive `TiltCard` components utilizing `framer-motion` to live-track mouse vectors and dynamically apply 3D rotational transformations in a preserved-3D rendering context.
- **🌒 Advanced Optical Glassmorphism:** Universal application of `.glass-panel` utilities utilizing heavy `backdrop-filter: blur`, dynamic radial neon gradients, and deep internal composite shadows to create a flawless physical glass illusion.
- **📡 Automatic Failover Orchestration (Portable Mode):** Backend logic engineered to intuitively fallback to an onboard JSON data layer when external Database clusters run into DNS/Connection constraints.
- **📄 The "Chameleon" Resume Engine:** React-governed resume screens utilizing `@media print` rules capable of morphing from a neon dark-glass UI instantly into a classic high-contrast, ATS-friendly corporate paper CV upon print command.
- **🛡️ Enterprise Verification Layer:** Intelligent modular popup modals enabling recruiters to securely access live commercial products and initiate pre-populated HR employment background validations seamlessly.

## 🛠️ Technology Stack

### Interactive Frontend
- **React.js 18:** Component-based UI logic.
- **Framer Motion:** Heavy computational math for spring physics and layout morphing.
- **Lucide-React:** Precision vector iconography.
- **MUI (Material UI):** Grid scaffolding under the custom CSS design system.
- **Vite:** High-speed optimized ES module bundling.

### Scalable Backend
- **Node.js & Express.js:** Fast routing and API generation.
- **MongoDB & Mongoose:** Cloud NoSQL data persistence layer.
- **Axios:** Asynchronous data fetching protocol.

## 🚀 Local Development Protocol

### 1. Source Acquisition
```bash
git clone https://github.com/mdyasar49/mern-portfolio-yasar.git
cd mern-portfolio-yasar
```

### 2. Backend Orchestration
```bash
cd server
npm install
# Configure .env: PORT=5001, MONGO_URI=your_cluster_uri
npm start
```

### 3. Frontend Instantiation
```bash
cd ../client
npm install
# Configure .env: REACT_APP_API_BASE_URL=http://localhost:5001/api
npm start
```
