### 💎 Core Architectural Pillars

#### 1. Hybrid Data Persistance Layer
The system implements a dual-mode data strategy. It prioritizes high-availability by seamlessly switching between a **MongoDB Cloud Instance** and a **Portable JSON Fallback**. This ensures the portfolio is always online, even during database maintenance.

#### 2. Advanced Resume Engine (Modular ATS)
Instead of static files, the portfolio features an internal API-driven resume engine.
- **Client-Side Generation**: Uses `html2pdf.js` for pixel-perfect document rendering.
- **Dynamic Injections**: Data is injected into specialized modules (Contact, Education, Experience) ensuring 100% data sync across the site and PDF.

#### 3. Cyber-Premium Design System
Built on a custom CSS-in-JS design system using **Material UI (MUI)**.
- **Glassmorphic UI**: High-blur background saturation and subtle neon borders.
- **Dynamic Motion**: Optimized framer-motion choreographies for a fluid user experience.

### 🚀 Quick Start Guide

#### Prerequisites
- Node.js (v16+)
- MongoDB Atlas Account (Optional - Fallback included)

#### Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/mdyasar49/mern-portfolio-yasar.git
   ```
2. **Setup Server**
   ```bash
   cd server
   npm install
   npm start
   ```
3. **Setup Client**
   ```bash
   cd ../client
   npm install
   npm start
   ```
